## COMANDOS PARA NAVEGAR

Para entrar dentro de uma pasta, use o comando **cd**
> cd nome_da_pasta
[comando cd](https://youtu.be/Soa5VwW09-w)

---

Para sair de dentro de uma pasta (voltar), use o comando **cd ..**
> cd ..
[comando cd..](https://youtu.be/LfHKCZVD1Vk)

---

Para ver os arquivos que está dentro da pasta, use o comando **ls**
> ls
[comando ls](https://youtu.be/PxYCr1UuI9M)

---

Para ver em qual pasta você está, use o comando **pwd**
> pwd
[comando pwd](https://youtu.be/vXOXg9eBC-w)

---

Para ver o conteudo dentro de um arquivo de texto .txt, use o comando **cat**
> cat arquivo_de_texto.txt
[comando cat](https://youtu.be/0KB3nRkfRxw)

---

Para carregar scripts (.sh), use o comando **source**
> source arquivo.sh
[comando source](https://youtu.be/P7qs23fMN2A)
